## Deployment Instructions

### Start Documentation Server
```bash
# Navigate to documentation directory
cd /opt/tower-echo-brain/docs

# Start the documentation server
python3 serve-docs.py 8001

# Access points:
# http://localhost:8001/              - Documentation hub
# http://localhost:8001/api/          - Interactive API docs
# http://localhost:8001/health        - Documentation health check
```

### Integration with Tower Ecosystem
```bash
# Add to nginx configuration for HTTPS access
location /docs/ {
    proxy_pass http://127.0.0.1:8001/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}

# Access via Tower dashboard
https://192.168.50.135/docs/
```

### Monitoring and Validation
```bash
# Run documentation validation
./validate-documentation.sh

# Check documentation health
curl http://localhost:8001/health

# Monitor documentation usage
curl http://localhost:8001/docs/status
```
